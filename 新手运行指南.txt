# 新闻聚合与分析系统 - 新手运行指南

大家好！欢迎使用这款新闻聚合与分析系统。别担心，即使你不太懂代码，按照下面的步骤也能轻松运行起来！

**前提条件：**

1.  **安装 Python:** 你需要先在你的电脑上安装 Python。推荐安装 Python 3.10 或更新版本。
    *   访问 Python 官网：[https://www.python.org/downloads/](https://www.python.org/downloads/)
    *   下载适合你操作系统的安装包（Windows、macOS 或 Linux）。
    *   **重要提示 (Windows 用户):** 在安装过程中，请务必勾选 "Add Python to PATH" 或类似的选项，这样在后续步骤中才能方便地使用命令。

2.  **下载项目文件:** 你需要获取这个项目的所有文件。
    *   **方式一 (推荐，如果安装了 Git):** 打开你的命令行工具（Windows 上是命令提示符(cmd)或 PowerShell，macOS/Linux 上是终端），进入你想存放项目的文件夹，然后运行：
        ```bash
        git clone [你的项目仓库地址]
        cd news_analyzer # 进入项目目录
        ```
        (请将 `[你的项目仓库地址]` 替换为实际的地址)
    *   **方式二 (直接下载):** 在项目的 GitHub 页面或其他来源，找到 "Download ZIP" 或类似的按钮，下载压缩包，然后解压到一个你方便找到的文件夹。

**运行步骤：**

1.  **打开命令行工具:**
    *   **Windows:** 按下 `Win` + `R` 键，输入 `cmd` 或 `powershell`，然后按 Enter。
    *   **macOS:** 打开“应用程序” -> “实用工具” -> “终端”。
    *   **Linux:** 通常可以在应用程序菜单中找到“终端”或使用快捷键（如 Ctrl+Alt+T）。

2.  **进入项目目录:** 使用 `cd` 命令进入你存放项目文件的那个文件夹。例如，如果你把项目放在了 D 盘的 `my_projects` 文件夹下，并且项目文件夹叫 `news_analyzer-main`，那么命令可能是：
    ```bash
    cd /d D:\my_projects\news_analyzer-main
    ```
    (请根据你的实际路径修改命令)

3.  **安装依赖库 (只需做一次):** 在命令行里运行下面的命令，它会自动下载并安装程序运行需要的各种工具库：
    ```bash
    pip install -r requirements.txt
    ```
    *   如果速度很慢或出错，可以试试国内的镜像源：
        ```bash
        pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
        ```
    *   这个过程可能需要几分钟，请耐心等待它完成。

4.  **运行程序:** 安装完依赖后，运行下面的命令启动程序：
    ```bash
    python main.py
    ```

5.  **开始使用:** 如果一切顺利，你应该能看到程序的界面弹出来了！

**常见问题：**

*   **`pip` 或 `python` 命令找不到?** 很可能是 Python 安装时没有添加到 PATH。重新安装 Python 并确保勾选了相关选项，或者搜索如何手动将 Python 添加到环境变量。
*   **安装依赖出错?** 检查网络连接，或者尝试更换镜像源。如果错误信息看不懂，可以截图寻求帮助。
*   **程序闪退或报错?** 检查命令行窗口是否有错误信息输出，这有助于定位问题。

祝你使用愉快！
